# RaspberryPico_Gameboy
Mini Taşınabilir El konsolu bu projede Temel olarak Raspberry Pi Pico ve 2.8inç SPI Dokunmatik TFT Ekran Modülü - 240x320 Piksel, ILI9341 Kullandım. Proje ile ilgili Kod ve devre şemasını paylaştım

Ben devreyi kablolardan kurtulmak için Delikli Bakır plaket üzerine lehimledim ve taşınabilir bir hale getirdim ve küçük bir Li-Po batarya ile enerji verdim devre şemasına eklemedim fakat eğer devrenize sizde batarya eklemek isterseniz raspbery pi pico güç besleme pini "VBUS" 40. Pin

# Dosyaları Raspberry Pi Pico'a Yükleme
